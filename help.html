Getting Started

Welcome to Physics Sandbox! Click "Start Simulating" on the home screen to begin.

Creating Objects

  • Circle: Click to spawn a circular object with random properties.
  • Rectangle: Click to spawn a rectangular object.
  • Ragdoll: Click to create a simple articulated figure.

Tools

  • Drag: Click and drag objects to move them (default behavior).
  • Delete: Select this tool, then click objects to remove them.
  • Impulse: Select this tool, then click objects to apply upward force.
  • Explosion: Select this tool, then click anywhere to create a radial explosion.

Controls

  • Gravity Toggle: Turn gravity on/off for zero-gravity simulations.
  • Grid Toggle: Show/hide a grid overlay for precise positioning.
  • Dark Mode: Switch between light and dark themes.
  • Screenshot: Capture and download an image of the current simulation.
  • Save/Load: Export your scene to JSON or import a saved scene.
  • Clear All: Remove all objects from the simulation.

Mobile Usage

On mobile devices, use the hamburger menu (‚ò∞) to access tools. Touch and drag to move objects. All tools work with touch input.

Tips

  • Experiment with different combinations of objects and forces.
  • Try creating chain reactions with explosions.
  • Use the grid for building structures.
  • Save interesting setups to load later.
