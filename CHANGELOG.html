# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Home screen with welcome message and start button
- Changelog page
- Improved CSS with gradients, animations, and better styling
- Backdrop blur effect for sidebar
- Links to documentation and changelog

### Changed
- Updated Matter.js usage to use Runner instead of deprecated Engine.run
- Enhanced button hover effects and transitions
- Improved color scheme with gradients

### Fixed
- ES module import issues by using global Matter object
- Touch support for mobile dragging
- Divergent branch syncing issues
- Save/load functionality for proper object restoration

### Technical
- Switched to Runner.run for engine execution
- Added home screen overlay before simulation starts
- Enhanced responsive design
