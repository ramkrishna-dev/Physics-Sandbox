Physics Sandbox is an open-source web application built with Matter.js, designed to provide an interactive and educational experience for exploring 2D physics simulations. Whether you're a student learning about gravity and collisions, or just someone who enjoys playing with virtual objects, this sandbox offers endless possibilities.

The project was created to demonstrate the power of modern web technologies in creating engaging, real-time simulations. It's fully responsive, works on both desktop and mobile devices, and encourages experimentation with physics concepts.

Built with love using HTML5, CSS3, JavaScript ES6+, and the amazing Matter.js physics engine.
